Chapitre 1 Introduction

1.1 Objectif

Le groupe State Of The R (SOTR) s’est réuni à Roscoff du 28 août 2017 au 1\(^{\mbox{er}}\) septembre 2017 pour une semaine d’auto-formation autour du logiciel R. Différents sujets ont été abordés durant cette semaine

  • Utilisation de git (surtout GitHub)
  • Automatisation et reproductibilité du code : rmarkdown ( Chapitres 2, 3, 4),
  • Manipulation de données :tidy (Chapitre 5)
  • Programmation orientée objets : R6 class (Chapitre 6),
  • Aspects techniques : Production de packages, Rcpp (Chapitre 7).

Nous avons travaillé avec R (3.4.1) et Rstudio (1.1.350) sur les plateformes linux (ubuntu version > 15.04), macosx (Sierra 10.12.6), windows (7).

Ce livre résume les expériences des participants durant l’atelier.
